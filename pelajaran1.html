HTML

 HTML (HyperText Markup Language) merupakan membuat dan menyusun bagian paragraf, heading, link atau tautan agar bisa menampilkan berbagai informasi di dalam sebuah browser web.

Cara Kerja HTML
Mari kita mulai pelajaran kita dengan HTML !
Aturan pertama penulisan code HTML adalah mengapit teks dengan tag. Tag dapat memberikan arti seperti judul atau tautan pada teks

Tag pembuka dan Tag Penutup
Sebagian besar element HTML memerlukan sepasang tag yaitu Tag pembuka dan Tag penutup. Dengan teks disisipkan di antara keduanya Tag pembuka <> dan Saat menggunakan tag penutup pastikan untuk meletakkan
  • Disetiap dokumen HTML, Anda harus memulai dengan deklarasi DOCTYPE. Ini akan memastikan versi HTML situs web Anda adalah yang terbaru :
    < !DOCTYPE html >
  • Pengcodean karakter :
    < meta charset="utf-8" >
  • Judul situs web :
    < title > Belajar Pemrograman < /title >
  • Tautan ke file CSS :
    < link rel="stylesheet" href="stylesheet.css" >
  • Membuat paragraf :
    < p > < /p >
  • Menulis komentar di HTML komentar harus diapit oleh :
    < !-- -->
  • Menetapkan gambar dengan menentukan url di atribut :
    < img src="url" >
  • Membuat daftar dengan bullet :
    < ul >
    < li > < /li >
    < li > < /li >
  • Membuat tautan URL tujuan :
    < a href="https://instagram.com/yenysyafitry" >Ig< /a >
  • Menampilkan gambar:
    < img src="URL">

CSS
CSS digunakan untuk mendesain situs web. Dengan CSS, Anda dapat mengubah hal-hal seperti warna, ukuran, dan spasi pada element HTML. Gambar disisi kiri hanya terdiri dari HTML saja. Dengan menerapkan CSS, Anda dapat menentukan style seperti tata letak, sesuai contoh yang ditampilkan disisi kanan
  • Menulis komentar di CSS komentar harus diapit oleh :
    /* */
  • Membuat Warna di CSS :
    color: #ff0000;
  • Membuat ukuran font :
    font-size: 20px;
  • Mengganti font :
    font-family: "Lucida Grande";
  • Mengubah warna latar belakang :
    background-color: #f3f372;
  • Mengubah lebar dan tinggi element :
    width: 200px;
    height: 80px;
